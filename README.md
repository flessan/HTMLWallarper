# 🖼️ HTML KDE Live Wallpaper Collection aka HTMLWallarper

[![License: LGPL](https://img.shields.io/badge/License-LGPL%20v.2.1-green.svg)](https://opensource.org/licenses/MIT)
[![KDE Plasma](https://img.shields.io/badge/Made%20for-KDE%20Plasma-blue?logo=kde&logoColor=white)](https://kde.org/)

A collection of lightweight, aesthetically pleasing, and performant Static HTML-based live wallpapers. Designed specifically for KDE Plasma users using the [HTML Wallpaper plugin.](https://github.com/MarcelRichter-GitHub/HTMLWallpaper) by [@MarcelRichter-GitHub](https://github.com/MarcelRichter-GitHub/)

---

## # +_= | System Prerequisites

In order for this wallpaper to work, you must install the **HTML Wallpaper** plugin and its supporting engine:

| Distro | Installation |
| :--- | :--- |
| **Arch / Based-on-Arch** | `sudo pacman -S qt6-webengine kdeplasma-addons` |
| **Void Linux** | `sudo xbps-install -S qt6-webengine` |
| **Debian / Ubuntu** | `sudo apt install qml6-module-qtwebengine plasma-widgets-addons` |
| **Fedora** | `sudo dnf install qt6-qtwebengine` |

**Plugin Source:** [HTML Wallpaper by the-unknown](https://store.kde.org/p/2137308)
